# Bundled Fonts

This project bundles **Source Han Sans SC Regular** for consistent Chinese UI rendering across machines.

- File: `SourceHanSansSC-Regular.otf`
- Source: Adobe Source Han Sans
- Upstream: https://github.com/adobe-fonts/source-han-sans
- License: SIL Open Font License 1.1

If you update this font file, keep the same filename or update `BUNDLED_FONT_PATHS` in `app_agent_desktop.py`.
